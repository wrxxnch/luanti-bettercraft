BlockHead Changelog
===================

0.1.0 (2024-07-02)
------------------

- Added the `--translate` and `--rotate` options for more flexible translation and rotation.
- Added the `--left` and `--right` options for easy rotation.
- Added the `--scale` option to resize blocks.
- Fixed `--backwards` turning the block upside-down.
- Fixed missing usage documentation.
- Fixed a crash on Mineclonia (reported by Robin213).
